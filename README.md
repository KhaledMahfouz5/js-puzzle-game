# JS Puzzle Game

A lightweight browser-based jigsaw puzzle game built with vanilla HTML, CSS, and JavaScript.

## Features
- Drag-and-drop puzzle pieces.
- Animated helper cursor that points from the next loose piece to its matching slot.
- Responsive layout that adapts to window resizing.

## Getting Started
### Run locally
1. From the repository root, start a local server:
   ```bash
   python -m http.server
   ```
2. Open the game in your browser:
   ```
   http://localhost:8000
   ```

## Project Structure
- `index.html` - Main game page.
- `assets/` - Styles, scripts, and images.
- `images/` - Puzzle artwork.
